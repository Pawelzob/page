Przykłady Funkcji w Pythonie

Home Page

1. Funkcja suma - dodawanie dwóch liczb

def suma(x:float, y:float) -> float:
    return (x + y)

print(suma(2, 4))
Wynik: 6

2. Funkcja pole_kola - pole koła

PI = 3.14
def pole_kola(PI:float, r:float) -> float:
    global x
    x = 5
    return PI * r * r

print(pole_kola(PI, 5))
print(x)
Wynik: 78.5
Wynik globalnej zmiennej x: 5

3. Funkcja witaj - powitanie

def witaj(imie:str, powitanie:str = 'siemano') -> str:
    wynik = suma(3, 4)
    return f"{powitanie}, {imie}"

print(witaj("Pablo", "Witaj krolu zloty"))
Wynik: Witaj krolu zloty, Pablo

4. Funkcja silnia - obliczanie silni

def silnia(n:int) -> int:
    if n == 0:
        return 1
    return n * silnia(n - 1)

print(silnia(5))
Wynik: 120

5. Funkcja pi - zwrócenie wartości liczby Pi

def pi() -> float:
    return 3.14

print(pi())
Wynik: 3.14

6. Funkcja suma_wielu - dodawanie wielu liczb

def suma_wielu(*args: int) -> int:
    return sum(args)

print(suma_wielu(5, 1, 5, 2, 5, 12, 5))
Wynik: 35

7. Funkcja celcjusz_to_fanhrenheit - przekształcanie Celsjusza na Fahrenheit

Tfah = 9 / 5
def celcjusz_to_fanhrenheit(C):
    return (C * Tfah + 32)

print(celcjusz_to_fanhrenheit(-20))
Wynik: -4.0

8. Funkcja filtr_l_parzystych - filtruj liczby parzyste

def filtr_l_parzystych(numbers: list[int]) -> list[int]:
    return [num for num in numbers if num % 2 == 0]

print(filtr_l_parzystych([2, 3, 4, 5]))
Wynik: [2, 4]
