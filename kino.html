1. Funkcja `print_seats`

def print_seats(seats: list): j = 1 for i in seats: if i == None: print(f"{j} Wolne") else: print(f"{j} Zajęte przez {i}") j += 1

Funkcja `print_seats` służy do wyświetlania wszystkich miejsc w systemie rezerwacji. Jej zadaniem jest sprawdzenie każdego miejsca w liście `seats`, aby zobaczyć, czy jest ono zajęte, czy wolne. Dla każdego miejsca wypisuje numer miejsca oraz informację o tym, czy jest ono wolne, czy zajęte przez konkretnego użytkownika.

2. Funkcja `add_reservation`

def add_reservation(seats: list): try: imie = str(input("Podaj imię: ")) numer = int(input("Podaj numer miejsca: ")) if numer > 10 or numer < 1: print("Numer jest poza zakresem (1-10)") return seats if seats[numer - 1] is None: seats[numer - 1] = imie print(f"Miejsce zostało zajęte przez {imie} na numerze {numer}") else: print("Zajęte") return seats except ValueError: print("Proszę podać prawidłowy numer")

Funkcja `add_reservation` pozwala na dodanie rezerwacji na wybrane miejsce. Użytkownik podaje swoje imię i numer miejsca, a funkcja sprawdza, czy miejsce jest dostępne (czyli czy jest puste). Jeśli tak, rezerwacja zostaje dodana. Jeśli nie, program informuje użytkownika, że dane miejsce jest już zajęte.

3. Funkcja `remove_reservation`

def remove_reservation(seats: list): try: numer = int(input("Podaj numer miejsca, które chcesz usunąć: ")) if numer > 10 or numer < 1: print("Numer jest poza zakresem (1-10)") return seats if seats[numer - 1] is not None: seats[numer - 1] = None print("Rezerwacja została usunięta") else: print("Miejsce nie ma rezerwacji") except ValueError: print("Proszę podać prawidłowy numer")

Funkcja `remove_reservation` pozwala na usunięcie rezerwacji z wybranego miejsca. Użytkownik podaje numer miejsca, które chce usunąć. Jeśli miejsce jest zarezerwowane, rezerwacja zostaje usunięta. Jeśli miejsce jest wolne, użytkownik dostaje informację, że nie ma rezerwacji do usunięcia.

5. Funkcja `check_availability`

def check_availability(seats: list): try: ile_numerow = int(input("Ile chcesz sprawdzić numerów: ")) for _ in range(ile_numerow): number = int(input("Podaj wybrane numery: ")) if seats[number - 1] is None: print(f"Miejsce {number} wolne") else: print(f"Miejsce zajęte przez: ", seats[number - 1]) except ValueError: print("Proszę podać prawidłowy numer")

Funkcja `check_availability` umożliwia użytkownikowi sprawdzenie dostępności wybranych miejsc. Użytkownik może wprowadzić numery miejsc, które chce sprawdzić, a program wypisuje, czy są one wolne, czy zajęte.

6. Funkcja `add_multiple_reservations`

def add_multiple_reservations(seats: list): try: while True: ile_miejsc = int(input("Ile miejsc chcesz zarezerwować: ")) for i in range(ile_miejsc): while True: numer = int(input(f"Podaj numer miejsca {i + 1}: ")) if seats[numer - 1] is None: imie = str(input("Podaj imię: ")) seats[numer - 1] = imie break # Zakończenie pętli, gdy miejsce zostało zarezerwowane else: print("To miejsce jest zajęte, wprowadź inne.") # Zapytanie, czy użytkownik chce dodać kolejne rezerwacje kontynuuj = input("Czy chcesz dokonać kolejnych rezerwacji? (tak/nie): ").lower() if kontynuuj != "tak": break # Zakończenie pętli, jeśli użytkownik nie chce więcej rezerwacji except ValueError: print("Wprowadziłeś nieprawidłowe dane. Spróbuj ponownie.") except IndexError: print("Podałeś numer miejsca spoza zakresu. Spróbuj ponownie.")

Funkcja `add_multiple_reservations` pozwala na zarezerwowanie wielu miejsc na raz. Program najpierw pyta, ile miejsc użytkownik chce zarezerwować, a następnie w pętli umożliwia wprowadzenie numeru miejsca i imienia rezerwującego.

Jeśli wybrane miejsce jest wolne (wartość `None`), zostaje ono zarezerwowane na imię podane przez użytkownika. Jeśli miejsce jest już zajęte, użytkownik jest proszony o podanie innego numeru.

Po dokonaniu rezerwacji użytkownik ma możliwość kontynuowania procesu rezerwacji lub zakończenia go, odpowiadając "tak" lub "nie" na pytanie.

Obsługiwane są błędy, takie jak nieprawidłowe dane wejściowe (np. nieprawidłowy numer miejsca) oraz próby wprowadzenia numeru spoza dostępnego zakresu.
